# Recommendation Service — Dynamic Rules

Проект реализует API для управления динамическими правилами рекомендаций продуктов.

## Реализованный функционал

### Динамические правила рекомендаций
Реализовано API для работы с динамическими правилами рекомендаций:

- Добавление правила
- Удаление правила
- Получение списка всех правил

Правила сохраняются в отдельной базе данных с использованием JPA и Liquibase.

---

## API

### Создание правила рекомендации
**POST /rule**

Пример запроса:
```json
{
  "productName": "Кредит наличными",
  "productId": "CASH_LOAN",
  "productText": "Кредит наличными с выгодной ставкой",
  "rules": [
    {
      "query": "age >",
      "arguments": ["18"],
      "negate": false
    },
    {
      "query": "income >",
      "arguments": ["30000"],
      "negate": false
    }
  ]
}

Ответ:

200 OK

Возвращается сохранённое правило с сгенерированным id

Получение списка правил

GET /rule

Ответ:

200 OK

Список всех сохранённых правил рекомендаций

Удаление правила

DELETE /rule/{id}

Ответ:

204 No Content
